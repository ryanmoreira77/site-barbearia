Projeto feito a uma barbearia com o intuito de ajudar  um microempreendedor, realizando prestação de serviços as novas pessoas no ramo dos negócios.

Tabela de preço e todas as informações foram coletadas junto ao Barbeiro, Gabriel.

Funcionalidades:
Página inicial moderna e visual atrativo
Apresentação da barbearia
Lista de serviços oferecidos
Integração com redes sociais
Acesso rápido ao Instagram
Design responsivo (desktop e mobile)
Interface estilizada para identidade masculina/barber style

Tecnologias utilizadas:
HTML5 — estrutura do site
CSS3 — estilização e layout
Google Fonts — tipografia personalizada
Responsividade com boas práticas de front-end

Objetivo do projeto
Este site foi desenvolvido com foco em:

Praticar desenvolvimento web
Aprimorar habilidades em HTML e CSS
Criar um projeto real para portfólio
Aplicar conceitos de design moderno
